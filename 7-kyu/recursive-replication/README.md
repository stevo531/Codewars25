<h2><a href=https://www.codewars.com/kata/57547f9182655569ab0008c4/train/javascript target="_blank">Recursive Replication</a></h2><h3>7 kyu</h3><p>You need to design a recursive function called <code>replicate</code> which will receive arguments <code>times</code> and <code>number</code>. </p><p>The function should return an array containing repetitions of the <code>number</code> argument. For instance, <code>replicate(3, 5)</code> should return <code>[5,5,5]</code>. If the <code>times</code> argument is negative, return an empty array.</p><p>As tempting as it may seem, do not use loops to solve this problem.</p>
